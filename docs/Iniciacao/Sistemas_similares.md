# Sistemas similares

## Sistemas de gestão de estágio (universidades)

- SAGU Sistema Acadêmico  
- TOTVS Educacional  

### O que eles fazem:
- Cadastro de estágio  
- Upload de documentos (TCE, plano)  
- Aprovação pela faculdade  

### Inspiração:
- Fluxo de aprovação (pendente → aprovado → rejeitado)  
- Validação manual (a ideia é automatizar)  


---

## Plataformas de estágio/emprego

- CIEE  
- Nube  

### O que eles fazem:
- Intermediação de estágio  
- Controle de contratos  
- Regras legais básicas  
- Regras da Lei do Estágio (carga horária, contrato, etc.)  
- Interface simples de cadastro  


---

## Camada jurídica (Lei 11.788)

### Valida automaticamente:
- Carga horária ≤ 6h/dia  
- Duração ≤ 2 anos  
- Tem TCE?  
- Tem seguro?  
- Bolsa (se não obrigatório)?  

### Saída:
- Regular  
- Risco de vínculo CLT

✔ Jurídico: OK

- Atividades não compatíveis com o curso
